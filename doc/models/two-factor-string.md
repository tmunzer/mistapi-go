
# Two Factor String

Request body for submitting a 2FA login code

*This model accepts additional fields of type interface{}.*

## Structure

`TwoFactorString`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `TwoFactor` | `string` | Required | One-time 2FA login code generated by the user's authenticator app |
| `AdditionalProperties` | `map[string]interface{}` | Optional | - |


# Mxedge Vm Params

Mist Edge VM parameters

## Structure

`MxedgeVmParams`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Model` | `*string` | Optional | SKU |
| `Name` | `*string` | Optional | User given name (optional) |
| `UserData` | `*string` | Optional | Base64 encoded user data |

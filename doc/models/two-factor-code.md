
# Two Factor Code

Request body for verifying a 2FA one-time password

*This model accepts additional fields of type interface{}.*

## Structure

`TwoFactorCode`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `TwoFactor` | `string` | Required | One-time password generated by the user's authenticator app |
| `AdditionalProperties` | `map[string]interface{}` | Optional | - |

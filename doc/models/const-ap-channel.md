
# Const Ap Channel

Country and regulatory-domain channel capabilities for AP radios

## Structure

`ConstApChannel`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Band2440mhzAllowed` | `*bool` | Optional | Whether 40 MHz channels are allowed on the 2.4 GHz band |
| `Band24Channels` | `map[string][]int` | Optional | Property key is the channel width |
| `Band24Enabled` | `*bool` | Optional | Whether the 2.4 GHz band is enabled for this country or regulatory domain |
| `Band5Channels` | `map[string][]int` | Optional | Property key is the channel width |
| `Band5Enabled` | `*bool` | Optional | Whether the 5 GHz band is enabled for this country or regulatory domain |
| `Band6Channels` | `map[string][]int` | Optional | Property key is the channel width |
| `Band6Enabled` | `*bool` | Optional | Whether the 6 GHz band is enabled for this country or regulatory domain |
| `Certified` | `*bool` | Optional | Whether AP operation is certified for this country or regulatory domain |
| `Code` | `*int` | Optional | Country code, ISO 3166-1 numeric |
| `DfsOk` | `*bool` | Optional | Whether DFS channels are allowed in this country or regulatory domain |
| `Key` | `*string` | Optional | Country code, in two-character |
| `Name` | `*string` | Optional | Country or regulatory-domain display name |
| `Uses` | `*string` | Optional | Regulatory ruleset used for AP channel validation |

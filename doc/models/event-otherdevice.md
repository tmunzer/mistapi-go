
# Event Otherdevice

Event generated by an other-device integration or attached device

## Structure

`EventOtherdevice`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `DeviceMac` | `*string` | Optional | Attached device MAC address associated with the other-device event |
| `Mac` | `*string` | Optional | Other-device MAC address associated with the event |
| `OrgId` | `*uuid.UUID` | Optional, Read-only | Unique identifier of a Mist organization |
| `SiteId` | `*uuid.UUID` | Optional, Read-only | Unique identifier of a Mist site |
| `Text` | `*string` | Optional | Human-readable message for the other-device event |
| `Timestamp` | `*float64` | Optional, Read-only | Epoch timestamp, in seconds |
| `Type` | `*string` | Optional | Event type for the other-device event |
| `Vendor` | `*string` | Optional | Manufacturer or vendor name reported for the other device |

## Example

```go
package main

import (
    "mistapi/models"
    "github.com/google/uuid"
)

func main() {
    eventOtherdevice := models.EventOtherdevice{
        DeviceMac:            models.ToPointer("device_mac6"),
        Mac:                  models.ToPointer("5c5b351e13b5"),
        OrgId:                models.ToPointer(uuid.MustParse("a97c1b22-a4e9-411e-9bfd-d8695a0f9e61")),
        SiteId:               models.ToPointer(uuid.MustParse("441a1214-6928-442a-8e92-e1d34b8ec6a6")),
        Text:                 models.ToPointer("Plugged: The Internal 5GB (SIM1) has been inserted into Internal 1."),
        Type:                 models.ToPointer("CELLULAR_EDGE_MODEM_WAN_PLUGGED"),
        Vendor:               models.ToPointer("cradlepoint"),
    }

}
```

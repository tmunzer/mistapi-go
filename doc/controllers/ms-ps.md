# Search Msp Org Group

Search MSP resources by query string. Currently `type=orgs` returns matching organizations in this MSP.

```go

    ctx context.Context,
    mspId uuid.UUID,
    mType models.MspSearchTypeEnum,
    q string,
    limit *int,
    start *string,
    end *string,
    duration *string,
    sort *string) (
    models.ApiResponse[models.ResponseSearch],
    error)
```

## Authentication

This endpoint requires [apiToken](../../doc/auth/custom-header-signature.md) **OR** [csrfToken](../../doc/auth/custom-header-signature-1.md)

## Parameters

| Parameter | Type | Tags | Description |
|  --- | --- | --- | --- |
| `mspId` | `uuid.UUID` | Template, Required | - |
| `mType` | [`models.MspSearchTypeEnum`](../../doc/models/msp-search-type-enum.md) | Query, Required | MSP search result type to return. enum: `orgs` |
